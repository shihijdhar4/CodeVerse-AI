## 📌 Overview

CodeVerse-AI is an AI-powered online code execution and testing platform that enables users to write, execute, and test code directly from a web browser. The platform provides an intuitive coding environment with real-time execution capabilities, making it suitable for students, developers, and coding enthusiasts.

The project is designed with a modern frontend, efficient backend architecture, and scalable technologies to deliver a seamless coding experience.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/shihijdhar4/CodeVerse-AI.git
```

---

## Move into Project

```bash
cd CodeVerse-AI
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm start
```

---

## Open Browser

```
http://localhost:4200
```
